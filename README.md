Este é um projeto de Gerenciador de Tarefas desenvolvido em Python para demonstração de conceitos de Programação Orientada a Objetos (POO), manipulação de arquivos JSON e organização modular de código.

Funcionalidades:

Criar Tarefas: Adiciona novas tarefas à lista.

Listar Tarefas: Exibe todas as tarefas com seu status atual (concluída ou pendente).

Concluir Tarefas: Altera o status de uma tarefa específica.

Remover Tarefas: Exclui tarefas da lista.

Persistência de Dados: Todas as tarefas são salvas automaticamente em um arquivo .json, garantindo que os dados não sejam perdidos ao fechar o programa.


obs:O projeto segue uma arquitetura modular

Como Executar?
Se certifique de ter Python instalado
Clone o repositorio
Execute main.py
